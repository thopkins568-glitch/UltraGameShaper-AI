# GameShaper-AI (v0.C — Ultra Starter)

GameShaper-AI is an LLM-powered prototype system that converts natural language prompts into
structured game specifications and scaffold JSON projects for 2D and 3D prototypes.

This repo is intentionally minimal but full-featured enough to iterate on a phone.
